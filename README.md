Food Recipe Application –



• Project Overview -

Food Recipe Application is a full-stack web application developed using React.js and Django REST Framework. It allows users to explore, search, and view detailed recipes with images, ingredients, and cooking instructions. The project demonstrates frontend and backend integration through REST APIs with a responsive user interface.



🛠 Technologies Used -

    • Frontend -
    
     -React.js
     -JavaScript (ES6)
     -HTML5 (JSX)
     -CSS3
     -Axios
     -React Router


    • Backend –
    
        -Python
        -Django
        -Django REST Framework
